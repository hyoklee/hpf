3.6 Web Page Installation

Simply point your web server's document root to hdf5perf/hdf5perfphp/hdf/. Some web servers are configured to look for index.php first instead of index.html. The main web page of HPF is hdf5perf/hdf5perfphp/hdf/index.html so you should browse it through http://your_web_server/index.html.
