About the Benchmarks

  • efc_no: Tests external file cache performance by opening multiple files through external links
  • cmpd_subset: Tests compound datatype subset selection and I/O performance
  • many_dsets: Tests performance with many datasets in a single file
  • vds: Tests virtual dataset (VDS) functionality and performance

Testing Framework: Catch2 with benchmarking enabled (10 samples per benchmark)
